Hello World

Skill: Compiling C++ Projects
Value 10pts

Objective

To compile a simple C++ program to learn the process of writing and running a program.

Materials

  1. C++ Compiler
  2. Access to your computer's terminal/command-line interface

Methods

  1. Open the terminal for your operating system.

  2. Type:

    g++ --version

    And press ENTER. Make sure that you get a message displaying what version of g++ you have. If not you will need to install this program

  3. Create a new file named hello-world.cpp and open the file for editing in your favorite basic text editor or IDE.

    Place the following code inside the file: 

    #include <iostream>
using namespace std;

int main()
{
  cout << "Hello World!\n";
  return 0;
}

    When you are finished, save the file

  4. Next we are going to compile your human-readable code file into a computer program your computer can understand. In your terminal, navigate to the location of your hello-world.cpp file. Once there, type:

    g++ hello-world.cpp

    If you don't see any error messages, then congratulations! You have successfully compiled your first C++ program. The g++ command turns your human-readable text file into binary. Just to be sure, look inside the directory where your hello-world.cpp file is. You should see an additional file named a.out. This new file is the binary executable file that the computer will read to run your computer program.

  5. Next, we are going to execute the binary file a.out. To do this, we simply need to type one more command. Please type:

    ./a.out

    You should see the words Hello World! print to the screen. If so, you have just completed the lab!

Conclusion

You now have a sense of the process for writing and compiling computer programs. You started by creating a file and filling it with code. Then you compiled the code into an executable. After this you ran the code and witnessed the output.
